# Occasions & Dress Codes

- Casual: denim, tees, sneakers; add one polished piece to elevate.
- Smart casual: clean sneakers or loafers, tailored pants, a knit or blazer.
- Business: solid colors, minimal patterns, structured layers.
- Evening: darker tones, richer fabrics, sharper contrast.
- Outdoor/cold: layer base + mid + shell; prioritize function then style.
