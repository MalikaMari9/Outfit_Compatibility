# Patterns Quick Guide

- Stripes: vertical stripes lengthen the silhouette; horizontal stripes add width.
- Checks: small checks feel casual; large checks feel bold and statement-like.
- Florals: large florals read dramatic; small florals read delicate and vintage.
- Mixing patterns: keep one pattern dominant, match at least one color, and vary scale.
- Texture + pattern: if a pattern is busy, keep textures simple (e.g., smooth knit).
- Pattern scale: mix a large-scale pattern with a small-scale pattern for balance.
- Pattern density: tight, busy patterns feel energetic; spaced patterns feel calmer.
- Directional patterns: verticals lengthen, diagonals add motion, horizontals add width.
- Color contrast in patterns: high-contrast patterns look louder; low-contrast patterns look subtle.
- Tone and saturation: muted patterns are easier to pair; saturated patterns should be the focal point.
- Pattern + solid rule: choose a solid color that appears in the pattern to unify the outfit.
